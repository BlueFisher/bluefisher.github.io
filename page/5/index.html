ж— жЁЎеһӢжҺ§еҲ¶ Model-Free Control

еҸ‘иЎЁдәҺ 2018-05-22 жӣҙж–°дәҺ 2018-12-30 еҲҶзұ»дәҺ Reinforcement Learning пјҢ Course by David Silver йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ

ж— жЁЎеһӢйў„жөӢ Model-Free Predication дёҖж–Үдёӯд»Ӣз»ҚдәҶеҰӮдҪ•еңЁжІЎжңүжЁЎеһӢзҡ„жғ…еҶөдёӢпјҢж №жҚ®дёҖдёӘзӯ–з•ҘжқҘиҜ„дј°иҜҘзӯ–з•Ҙзҡ„д»·еҖјеҮҪж•°пјҢиҖҢиҝҷзҜҮж–Үз« е°ҶеңЁдёҠж–Үзҡ„еҹәзЎҖдёҠпјҢжқҘдјҳеҢ–д»·еҖјеҮҪж•°гҖӮ

жҲ‘д»¬еҸҜд»ҘеңЁж— жЁЎеһӢзҡ„жғ…еҶөдёӢдҪҝз”Ёд№ҹеҸҜд»ҘеңЁжңүжЁЎеһӢдҪҶжҳҜжЁЎеһӢеӨӘеәһеӨ§еҸӘиғҪз”ЁйҮҮж ·зҡ„ж–№ејҸжқҘиҝӣиЎҢжҺ§еҲ¶гҖӮ

дјҳеҢ–жҺ§еҲ¶еҸҜд»ҘеҲҶдёәдёӨзұ»еӯҰд№ ж–№ејҸпјҡ

  1. еҗҢзӯ–з•ҘеӯҰд№  (On-policy learning) пјҡжҲ‘д»¬йңҖиҰҒдјҳеҢ–дёҖдёӘзӯ–з•Ҙ \(\pi\) пјҢе°ұеҝ…йЎ»иҝӣиЎҢйҮҮж ·пјҢиҖҢйҮҮж ·зҡ„ж–№ејҸе°ұжҳҜж №жҚ®иҰҒеӯҰд№ зҡ„зӯ–з•Ҙ \(\pi\) гҖӮиҜҘз§ҚеӯҰд№ ж–№ејҸз§°дёәеҗҢзӯ–з•ҘпјҢеҚійҮҮж ·дёҺеӯҰд№ зҡ„дёәеҗҢз§Қзӯ–з•ҘгҖӮ
  2. ејӮзӯ–з•ҘеӯҰд№  (Off-policy learning) пјҡеҗҢж ·жҳҜдјҳеҢ–зӯ–з•Ҙ \(\pi\) пјҢдҪҶйҮҮж ·ж—¶йҒөеҫӘзҡ„зӯ–з•ҘеҚҙжҳҜеҸҰдёҖз§Қзӯ–з•Ҙ \(\mu\) пјҢзӣёеҪ“дәҺжҳҜж №жҚ®еүҚдәәе·Іжңүзҡ„з»ҸйӘҢ \(\mu\) жқҘеӯҰд№ зҺ°жңүзӯ–з•Ҙ \(\pi\) пјҢвҖңз«ҷеңЁе·Ёдәәзҡ„иӮ©иҶҖдёҠвҖқгҖӮиҜҘз§ҚеӯҰд№ ж–№ејҸжҲҗдёәејӮзӯ–з•ҘпјҢеҚійҮҮж ·дёҺеӯҰд№ дёәдёҚеҗҢзҡ„зӯ–з•ҘгҖӮ
йҳ…иҜ»е…Ёж–Ү »

еҹәдәҺжЁЎеһӢзҡ„еҠЁжҖҒи§„еҲ’ Planning by Dynamic Programming

еҸ‘иЎЁдәҺ 2018-05-19 жӣҙж–°дәҺ 2018-12-30 еҲҶзұ»дәҺ Reinforcement Learning пјҢ Course by David Silver йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ

ејәеҢ–еӯҰд№ зҡ„жңҖз»Ҳзӣ®зҡ„е°ұжҳҜжүҫеҲ° MDP зҡ„жңҖдјҳзӯ–з•ҘпјҢиҖҢеңЁе·ІзҹҘжЁЎеһӢзҡ„жғ…еҶөдёӢпјҢйҖҡеёёз”ЁеҠЁжҖҒи§„еҲ’ (Dynamic Programming) ж–№жі•жқҘеҜ»жүҫжңҖдјҳзӯ–з•ҘгҖӮ

еҠЁжҖҒи§„еҲ’жҳҜдёҖз§Қйқһеёёеёёи§Ғзҡ„з®—жі•пјҢиҰҒеә”з”ЁеҠЁжҖҒи§„еҲ’ж–№жі•йңҖиҰҒж»Ўи¶ідёӨдёӘжқЎд»¶пјҡ

  1. еҺҹжң¬еӨҚжқӮзҡ„й—®йўҳеҸҜд»Ҙиў«еҲҶи§ЈжҲҗзӣёеҜ№з®ҖеҚ•еӯҗй—®йўҳпјҢжұӮеҮәжңҖдјҳеӯҗй—®йўҳе°ұжұӮеҮәдәҶжңҖдјҳи§Ј
  2. еӯҗй—®йўҳдјҡеҸҚеӨҚеҮәзҺ°еӨҡж¬ЎпјҢе№¶дё”еӯҗй—®йўҳзҡ„з»“жһңеҸҜд»Ҙиў«еӯҳеӮЁиө·жқҘз»§иҖҢи§ЈеҶіеҗҺз»ӯзҡ„й—®йўҳ

иҖҢ MDP жӯЈеҘҪж»Ўи¶ідәҶдёҠиҝ°дёӨдёӘзү№жҖ§пјҡ

  1. иҙқе°”жӣјж–№зЁӢзӣёеҪ“дәҺе°ҶеӨҚжқӮй—®йўҳеҲҶи§ЈдёәдәҶйҖ’еҪ’зҡ„еӯҗй—®йўҳ
  2. д»·еҖјеҮҪж•°зӣёеҪ“дәҺеӯҳеӮЁдәҶд№ӢеүҚеӯҗй—®йўҳзҡ„и§Ј

йҖҡиҝҮеҲӨж–ӯдёҖдёӘзӯ–з•Ҙзҡ„д»·еҖјеҮҪж•°пјҢз”ЁиҜҘзӯ–з•ҘдёӢзҡ„жңҖдјҳд»·еҖјеҮҪж•°жқҘжӣҙж–°зӯ–з•Ҙе№¶дёҚж–ӯйҮҚеӨҚдёҠиҝ°иҝҮзЁӢзӣҙиҮіжүҫеҲ°жңҖдјҳд»·еҖјеҮҪж•°дёҺжңҖдјҳзӯ–з•ҘдҫҝжҳҜеҠЁжҖҒи§„еҲ’зҡ„ж ёеҝғжҖқжғігҖӮ

еңЁжұӮи§Ј MDP зҡ„жңҖдјҳзӯ–з•ҘдёӯдҪҝз”ЁеҠЁжҖҒи§„еҲ’дёҖиҲ¬еҸҜд»Ҙз”ЁжқҘпјҡ

  • йў„жөӢпјҡиҫ“е…Ҙ MDP \(<\mathcal{S},\mathcal{A}, \mathcal{P}, \mathcal{R}, \gamma>\) е’Ңзӯ–з•Ҙ \(\pi\) пјҢжҲ–иҫ“е…Ҙ MRP \(<\mathcal{S}, \mathcal{P}^\pi, \mathcal{R}^\pi, \gamma>\) пјҢиҫ“еҮәд»·еҖјеҮҪж•° \(v_\pi\)
  • жҺ§еҲ¶пјҡиҫ“е…Ҙ MDP \(<\mathcal{S},\mathcal{A}, \mathcal{P}, \mathcal{R}, \gamma>\) пјҢиҫ“еҮәжңҖдјҳд»·еҖјеҮҪж•° \(v_*\) е’ҢжңҖдјҳзӯ–з•Ҙ \(\pi_*\)
йҳ…иҜ»е…Ёж–Ү »

ж— жЁЎеһӢйў„жөӢ Model-Free Predication

еҸ‘иЎЁдәҺ 2018-05-19 жӣҙж–°дәҺ 2018-12-30 еҲҶзұ»дәҺ Reinforcement Learning пјҢ Course by David Silver йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ

еңЁе·ІзҹҘжЁЎеһӢзҡ„жғ…еҶөдёӢпјҢеҸҜд»Ҙз”ЁеҠЁжҖҒи§„еҲ’ж–№жі•и§ЈеҶі MDP й—®йўҳпјҢдҪҶеңЁејәеҢ–еӯҰд№ дёӯпјҢжӣҙеӨҡйҒҮеҲ°зҡ„жҳҜж— жЁЎеһӢжғ…еҶө (Model-Free) пјҢд№ҹе°ұжҳҜжЁЎеһӢ \(P^a_{ss'}\) жҳҜжңӘзҹҘзҡ„пјҢжҜ”еҰӮдёҖжһ¶зӣҙеҚҮжңәеӨ„дәҺ \(s\) зҠ¶жҖҒпјҢеңЁйҮҮеҸ–еҠЁдҪң \(a\) д№ӢеҗҺпјҢжҲ‘д»¬дёҚзҹҘйҒ“жңүеӨҡеӨ§жҰӮзҺҮдјҡиҪ¬з§»еҲ° \(s'\) зҠ¶жҖҒпјҢеҪ“ж—¶зҡ„зҺҜеўғзҠ¶еҶөдјҡдә§з”ҹеҫҲеӨ§зҡ„е№Іжү°гҖӮ

жң¬ж–Үе°ұжҳҜиҰҒеңЁеңЁжңӘзҹҘжЁЎеһӢзҡ„жғ…еҶөдёӢпјҢдј°и®Ўд»·еҖјеҮҪж•°пјҢеҢ…жӢ¬и’ҷзү№еҚЎзҪ—ж–№жі•дёҺж—¶й—ҙе·®еҲҶж–№жі•гҖӮ

еҸ‘иЎЁдәҺ 2018-05-17 жӣҙж–°дәҺ 2018-12-30 еҲҶзұ»дәҺ Reinforcement Learning йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ

еңЁ DeepMind зҡ„ Asynchronous Methods for Deep Reinforcement Learning дёҖж–ҮдёӯпјҢд»Ӣз»ҚдәҶеҸҰдёҖз§Қз®ҖеҚ•зҡ„иҪ»йҮҸзә§ж·ұеәҰејәеҢ–еӯҰд№ жЎҶжһ¶вҖ”вҖ”ејӮжӯҘең°иҝӣиЎҢж·ұеәҰзҘһз»ҸзҪ‘з»ңжўҜеәҰдёӢйҷҚгҖӮиҜҘжЎҶжһ¶еҸҜд»ҘйҖӮз”ЁдәҺи®ёеӨҡејәеҢ–еӯҰд№ з®—жі•пјҢи®әж–Үдёӯе°ҶжЎҶжһ¶еҘ—еңЁдәҶеӣӣз§Қж ҮеҮҶејәеҢ–еӯҰд№ з®—жі•дёҠпјҢеҲҶеҲ«жҳҜ one-step Q-Learning, one-step Sarsa, n-step Q-Learning е’Ң Advantage Actor-Critic пјҢйғҪеҸ–еҫ—дәҶдёҚй”ҷзҡ„зЁіе®ҡж•ҲжһңгҖӮе…¶дёӯжңҖеҗҺдёҖдёӘ Asynchronous Advantage Actor-Critic (A3C) з®—жі•жӣҙжҳҜжңүзқҖжңҖеҘҪзҡ„жҖ§иғҪиЎЁзҺ°пјҢеңЁеӨҡж ё CPU дёҠеӯҰд№ жҜ”еңЁ GPU дёҠеӯҰд№ зҡ„ж—¶й—ҙеҮҸе°‘дәҶдёҖеҚҠпјҢжң¬ж–Үд№ҹдјҡйҮҚзӮ№е…іжіЁ A3C з®—жі•гҖӮ

еҸ‘иЎЁдәҺ 2018-05-16 жӣҙж–°дәҺ 2018-12-30 еҲҶзұ»дәҺ Reinforcement Learning йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ

Deep Deterministic Policy Gradient (DDPG) з®—жі•еҮәиҮӘи®әж–Ү Continuous control with deep reinforcement learning пјҢиҜҘз®—жі•зҡ„жҖқжғіеҫҲз®ҖеҚ•пјҢе°ұжҳҜе°Ҷ зЎ®е®ҡжҖ§зӯ–з•ҘжўҜеәҰ Deterministic Policy Gradient дёҺ Deep Q-Network дёӨдёӘз®—жі•з»“еҗҲиө·жқҘпјҢи§ЈеҶідәҶ DQN еҸӘиғҪиҝҗз”ЁеңЁзҰ»ж•ЈиЎҢдёәз©әй—ҙдёҠзҡ„еұҖйҷҗпјҢеҗҢж—¶еҖҹйүҙ DQN зҡ„зҘһз»ҸзҪ‘з»ңгҖҒз»ҸйӘҢеӣһж”ҫе’Ңи®ҫзҪ® target зҪ‘з»ңдҪҝ DPG дёӯзҡ„ Actor-Critic з®—жі•жӣҙе®№жҳ“ж”¶ж•ӣ

Deterministic Policy Gradient

еҸ‘иЎЁдәҺ 2018-05-16 жӣҙж–°дәҺ 2018-12-11 еҲҶзұ»дәҺ Reinforcement Learning йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ

еңЁ Deterministic Policy Gradient Algorithms дёҖж–ҮдёӯпјҢDeepMind еңЁеҺҹжңүйҡҸжңәзӯ–з•ҘжўҜеәҰ (Stochastic Policy Gradient) з®—жі•зҡ„еҹәзЎҖдёҠд»Ӣз»ҚдәҶзЎ®е®ҡжҖ§зӯ–з•ҘжўҜеәҰз®—жі• (Deterministic Policy Gradient Algorithms DPG) жқҘи§ЈеҶіиҝһз»ӯжҖ§еҶізӯ–й—®йўҳпјҢжҳҜд№ӢеҗҺ Deep DPG (DDPG) з®—жі•зҡ„еҹәзЎҖгҖӮ

дј з»ҹзҡ„зӯ–з•ҘжўҜеәҰз®—жі•д»ҘжҰӮзҺҮеҪўејҸ \(\pi_\theta(a|s) = \mathbb{P}[a|s; \theta]\) жқҘиЎЁзӨәдёҖдёӘзӯ–з•ҘпјҢд»ҘжӯӨжқҘйҡҸжңәзҡ„йҖүжӢ©иЎҢдёәгҖӮдҪҶ DPG з”ЁдёҖз§ҚзЎ®е®ҡжҖ§зҡ„зӯ–з•ҘеҪўејҸ \(a=\mu_\theta(s)\) гҖӮ

DPG жңүзқҖжҜ” PG жӣҙз®ҖеҚ•зҡ„еҪўејҸпјҡиЎҢдёәд»·еҖјеҮҪж•°зҡ„жңҹжңӣпјҢиҝҷд№ҹдҪҝеҫ— DPG жҜ” PG жӣҙеҠ жңүж•ҲпјҢеҗҢж—¶еңЁй«ҳз»ҙеәҰиЎҢдёәз©әй—ҙдёӯд№ҹжҜ” PG иЎЁзҺ°еҫ—жӣҙеҠ еҘҪгҖӮ
